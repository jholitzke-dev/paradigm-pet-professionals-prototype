
# Paradigm Pet Professionals Website Redesign

## Project Overview

This project is a UX/UI redesign of the Paradigm Pet Professionals website, a pet advice and information site designed to help users find guidance on cats, dogs, small animals, and general pet care resources.

The original website had usability, layout, and visual design issues that made it difficult for users to navigate and find helpful information. I redesigned the site to create a cleaner, more organized, and more user-friendly experience for pet owners seeking reliable pet care guidance.

## Project Goal

The goal of this project was to improve the overall user experience of the Paradigm Pet Professionals website while meeting stakeholder specifications for layout, navigation, branding, and content organization.

The redesigned site helps users find pet care information more easily while presenting the content in a more professional, accessible, and user-friendly format.

## Stakeholder Specifications

This redesign was created based on stakeholder specifications for the Paradigm Pet Professionals website. The project required improving the site’s usability, organization, visual design, and content structure while maintaining the purpose of the website as a pet advice and information resource.

The redesign focused on meeting stakeholder expectations by creating a clearer navigation structure, improving access to pet care resources, applying consistent branding, and making the site easier for users to understand and use.

## Design Process

I followed a UX/UI design process that included reviewing stakeholder specifications, identifying usability issues in the original website, planning the site structure, creating wireframes in Figma, and building the final prototype.

The wireframing stage helped me plan the layout, navigation, page structure, and content placement before moving into the final visual design and front-end development.

## Pages Included

* **Home** - Introduces the site and provides general pet care resources.
* **Cats** - Provides cat care information and helpful resources.
* **Dogs** - Provides dog care, nutrition, vaccination, and behavior-related information.
* **Small Animals** - Includes care resources for small pets such as guinea pigs, hamsters, mice, chinchillas, gerbils, and hermit crabs.
